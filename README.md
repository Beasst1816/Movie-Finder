# Vibe Match Movies

**Vibe Match Movies** is a modern Flutter application designed to help users discover films that match their current mood or "vibe." Built with a sleek, dark-themed interface, the app provides an intuitive and immersive movie-browsing experience.

## Features

*   **Vibe-Based Discovery:** Find movies tailored to your specific mood or preference.
*   **Modern UI:** A clean, dark-mode design optimized for a cinematic feel (Dark Theme).
*   **Cross-Platform:** Built with Flutter for a consistent experience on both iOS and Android.
*   **Battery Status Integration:** Monitor your device's battery while browsing.

## Getting Started

### Prerequisites

*   Flutter SDK
*   Android Studio / VS Code

### Installation

1.  Clone the repository:
    ```sh
    git clone https://github.com/YOUR_USERNAME/moviefinder.git
    ```
2.  Navigate to the project directory:
    ```sh
    cd moviefinder
    ```
3.  Install dependencies:
    ```sh
    flutter pub get
    ```
4.  Run the application:
    ```sh
    flutter run
    ```
